| Ctrl+a | 光标移动到开始位置 |
| Ctrl+e | 光标移动到最末尾 |
| Ctrl+k | 删除此处至末尾的所有内容 |
| Ctrl+u | 删除此处至开始的所有内容 |
| Ctrl+w | 删除此处 之前的单词 |
| Ctrl+Left/Right-Arrow | 按单词左右移动 |
| Shift+PgUp/PgDn | 终端上下翻页滚动 |
| Shift+Ctrl+N | 打开新的终端窗口 |
| Shift+Ctrl+T | 打开新的标签页 |
| Shift+Ctrl+W | 关闭标签页 |
| Shift+Ctrl+C | 复制 |
| Shift+Ctrl+V | 粘贴 |
| Alt+数字 | 切换至对应的标签页 |
